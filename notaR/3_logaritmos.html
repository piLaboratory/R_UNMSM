En este ejercicio vamos a añadir nuevas variables al data frame islas que hás creado en el ejercício Leyendo Datos.

  1. Inicia tu script R con el código que has utilizado para resolver el ejercicio Leyendo Datos. Este código debe crear el objeto islas en tu espacio de trabajo. Importante: si no creas ese objeto islas verás el terrible mensaje de error "Ocorreu um erro na execução do seu código!"

  2. Añade una variable en el objeto islas que contenga el logaritmo natural de la variable species.richness. Esa nueva variable debe llamarse species.richness.log.

  3. Añade una otra variable en el objeto islas que contenga el logaritmo natural de la variable island.area. Esa nueva variable debe llamarse especie.riqueza.log.
